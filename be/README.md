## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

# 環境構築（要改善）

```
# コンテナ立ち上げ
docker comopse up -d

# apiコンテナに入る
docker exec -it oshizukani-api sh

# パッケージのインストール（コンテナの中で行う）
yarn install

# マイグレーションの実行（コンテナの外で行う）
yarn migration:run

# サーバー起動（コンテナの中で行う）
yarn start
```

# DB 操作

```
# dbコンテナに入る
docker exec -it be_db_1 sh
or
docker exec -it be_db_1 bash

# ユーザーを指定してdbに接続
psql -U postgres

# dbの一覧確認
\l

# db選択
\c oshizukani-dev

# テーブル一覧確認
\dt

# テーブルのカラム確認
\d テーブル名
ex.
\d user
```

## migration

```
# migrationファイルの作成（entityファイルから作る）
yarn migration:gen src/migration/****
```
